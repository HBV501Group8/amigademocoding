AsmOne Quick Command Reference

General

  • ESC — Return to editor
  • RETURN — Execute current line / command
  • TAB — Toggle editor ↔ CLI
  • HELP — Show available commands

Assembling & Running

  • A — Assemble current source
  • AA — Assemble & Run automatically
  • R — Run last assembled program
  • G — Go (run at specific address)
  • T — Trace (single-step execution)
  • B — Set breakpoint

Editing & Source

  • E — Edit source
  • F — Find text
  • L — Load source
  • S — Save source
  • N — New (clear buffer)

Disk Operations

  • DIR — List directory
  • LOAD <file> — Load binary to memory
  • SAVE <file> — Save memory to disk
  • DEL <file> — Delete file

Memory & Debugging

  • D <addr> — Dump memory
  • M <addr> — Modify memory
  • REG — Show registers
  • SET — Change register values
  • BR <addr> — Breakpoint at address
  • C — Continue after break

Assembler Directives

  • DC.B / DC.W / DC.L — Define constant
  • DS.B / DS.W / DS.L — Reserve storage
  • EQU — Define constant value
  • INCLUDE — Include another file
  • ORG — Set program load address

Example flow: E → edit code → ESC → A assemble → R run.
