Amiga OCS / ECS — Register Map

Modular reference: Agnus (DMA/Copper/Blitter), Denise (Video), Paula (Audio/I/O). Use search to filter by name or address.

Agnus — DMA, Blitter, Copper, Memory Control

Registers used for DMA control, blitter operations, copper pointer, and memory arbitration.

AddressNameBitsDescription
$DFF000INTENA15–0Interrupt enable (OCS/ECS). Master SET/CLR and per-source enables.
$DFF004INTREQ15–0Interrupt request / control (write 1 to clear in some models).
$DFF006DSKSYNC15–0Disk sync — used for disk timing.
$DFF008JOY0DAT15–0Joystick 0 / direction/buttons.
$DFF00AJOY1DAT15–0Joystick 1 / direction/buttons.
$DFF00CCLXCON15–0Collision control (ECS/AGA variations exist).
$DFF012POT0DAT15–0Potentiometer / paddle data 0.
$DFF014POT1DAT15–0Potentiometer / paddle data 1.
$DFF016ADKCON15–0Audio/DSP control (DMA pace & stereo options on some models).
$DFF01ASDMCTL15–0Serial DMA control and parallel I/O flags.
$DFF020CLXCON215–0Additional collision control (ECS).
$DFF028INTMSK (alias)15–0Alternate interrupt mask locations on some systems.
$DFF040BLTCON015–0Blitter control register 0 — function, logic/masking, shift etc.
$DFF042BLTCON115–0Blitter control register 1 — modulo and minterms.
$DFF044BLTAFWM15–0Blitter destination first word mask.
$DFF046BLTALWM15–0Blitter destination last word mask.
$DFF048BLTCPTH31–0Blitter A pointer (high/low parts across registers).
$DFF050BLTDPTH31–0Blitter D pointer (destination) high/low.
$DFF058BLTSIZE15–0Blitter size (Y, X) words specifying height and width of operation.
$DFF060COP1LC31–0Copper list pointer (address of first copper instruction).
$DFF070COPJMP115–0Copper immediate jump — used to control Copper flow (ECS/AGA differences).
$DFF076DMACON15–0DMA control register: master enable, control bits for blitter, copper, bitplanes, sprites, audio channels etc.
$DFF07AADKCON215–0Audio/DMA additional control (ECS extensions).

Denise — Video, Bitplanes, Sprites, Palette

Registers controlling display timing, bitplane pointers, sprites, colors, and playfield/window parameters.

AddressNameBitsDescription
$DFF080DIWSTRT15–0Display window start — vertical/horizontal start positions.
$DFF082DIWSTOP15–0Display window stop — vertical/horizontal end positions.
$DFF084DDFSTRT15–0Data fetch start — where memory fetches begin.
$DFF086DDFSTOP15–0Data fetch stop — end of fetch window.
$DFF088DMACONR15–0DMA control (read) — status mirror of DMACON.
$DFF08EBPLCON015–0Bitplane control 0 — planes count, HAM, hires, etc.
$DFF090BPLCON115–0Bitplane control 1 — scrolling, priorities.
$DFF092BPLCON215–0Bitplane control 2 — sprite collision options (ECS additions).
$DFF0A0BPL1PTH / BPL1PTL31–0Bitplane 1 pointer (high/low) — pointer to bitmap data in chip RAM.
$DFF0E0BPL2PTH / BPL2PTL31–0Bitplane 2 pointer.
$DFF100COLOR0011–0Palette register 0 (12-bit RGB: RRRR GGGG BBBB). COLOR00–COLOR31 contiguous.
$DFF120SPR0POS15–0Sprite 0 position and control (X,Y and enable bits across registers).
$DFF140SPR0CTL15–0Sprite control register: priority, expansion, collision enable.
$DFF180COLOR3111–0Palette register 31 (end of palette block).

Paula — Audio channels, Disk, I/O

Audio channel sample pointers, lengths, periods, volumes, and I/O ports like disk and joystick registers.

AddressNameBitsDescription
$DFF0A0AUD0LCH / AUD0LCL31–0Audio 0 sample pointer (high/low words).
$DFF0A4AUD0LEN15–0Audio 0 sample length (in words).
$DFF0A6AUD0PER15–0Audio 0 period (playback frequency control).
$DFF0A8AUD0VOL6–0Audio 0 volume (0-64 typical range, 7-bit).
$DFF0B0AUD1LCH / AUD1LCL31–0Audio 1 sample pointer.
$DFF0B4AUD1LEN15–0Audio 1 length.
$DFF0B6AUD1PER15–0Audio 1 period.
$DFF0B8AUD1VOL6–0Audio 1 volume.
$DFF0D0DSKDATAL15–0Disk DMA data register (low).
$DFF0D4DSKPT31–0Disk DMA pointer/position registers for track reading.
$DFF0F0INTREQ15–0Interrupt request (mirror/alias for INTREQ area).
