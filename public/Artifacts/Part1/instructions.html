M68000 Instruction Set

Data Movement
MOVE – Copy data from source to destination.
MOVEA – Move data into an address register.
MOVEP – Move data between memory and register with padding (useful for peripheral I/O).
MOVEQ – Move immediate quick value (8-bit) into a data register.
LEA – Load effective address into an address register.
PEA – Push effective address onto the stack.
Arithmetic
ADD – Add source to destination.
ADDA – Add to address register.
ADDI – Add immediate to destination.
ADDQ – Add quick immediate (small constant).
SUB – Subtract source from destination.
SUBQ – Subtract quick immediate.
MULS – Signed multiply.
MULU – Unsigned multiply.
DIVS – Signed divide.
DIVU – Unsigned divide.
Logical / Bit Operations
AND – Logical AND.
OR – Logical OR.
EOR – Exclusive OR.
NOT – Logical NOT.
BTST – Test a bit.
BSET – Set a bit.
BCLR – Clear a bit.
BCHG – Change (toggle) a bit.
Branching & Control
BRA – Branch always.
BSR – Branch to subroutine.
Bcc – Conditional branch (e.g., BEQ, BNE, BGT).
DBcc – Decrement and branch if condition true.
JMP – Jump to address.
JSR – Jump to subroutine.
RTS – Return from subroutine.
RTE – Return from exception.
System / Stack
NOP – No operation.
RESET – Reset external devices.
STOP – Halt until interrupt.
TRAP – Cause a software interrupt (trap).
LINK – Create stack frame for subroutine.
UNLK – Remove stack frame.
TST – Test operand (set condition codes).
Addressing Modes
Immediate – Operand is part of the instruction. Example: MOVE #5,D0
Data Register Direct – Operand in a data register. Example: MOVE D0,D1
Address Register Direct – Operand in an address register. Example: MOVEA A0,A1
Address Register Indirect – Operand at memory location pointed by address register. Example: MOVE (A0),D0
Postincrement – Use then increment. Example: MOVE (A0)+,D0
Predecrement – Decrement then use. Example: MOVE -(A0),D0
Displacement – Base register + offset. Example: MOVE 16(A0),D0
Indexed – Base register + index + displacement. Example: MOVE (A0,D1),D0
Absolute Short – Direct 16-bit absolute address. Example: MOVE $1234,D0
Absolute Long – Direct 32-bit absolute address. Example: MOVE $12345678,D0
PC-Relative – Relative to PC. Example: MOVE 8(PC),D0
