Flat shading is a lighting and coloring technique where each surface is rendered using a single, uniform color—rather than smooth gradients or blended transitions.

The Core Idea

Imagine an object as a collection of distinct flat faces (planes). Each face is assigned one color based on how it is oriented relative to a light source, with no soft blending between surfaces.

How It Works

  • Choose a light source: For example, light coming from the top-left.
  • Divide the object into planes: Even curved objects are approximated as flat sections.
  • Assign one color per plane:
    • Faces toward the light → lighter color
    • Faces away from the light → darker color
    • Side angles → midtones
  • No gradients or blur: Transitions between light and shadow are sharp and clearly defined.

How Is Color Calculated?

The color of each face depends on the angle between its normal vector (a line perpendicular to the surface) and the direction of the light source. Faces that point more directly toward the light appear brighter, while those angled away appear darker.

To calculate this, we use the dot product between the face’s normal vector and the light direction. Since we already have the normal vectors for each face of the cube (from hidden line rendering), this allows us to easily determine how much light each face receives.

normal_Z:
	movem.l	d1-d6,-(sp)

	move.w	0(a2),d6		; coord 1
	muls	#6,d6		; *4
	move.w	(a3,d6.w),d0	; X1
	move.w	2(a3,d6.w),d1	; Y1

	move.w	4(a2),d6	; coord 2
	muls	#6,d6		; *4
	move.w	(a3,d6.w),d2	; X2
	move.w	2(a3,d6.w),d3	; Y2

	move.w	2(a2),d6	; coord 3
	muls	#6,d6		; *4
	move.w	(a3,d6.w),d4	; X3
	move.w	2(a3,d6.w),d5	; Y3

	sub.l	d4,d0		; (X1-X3)=Vx1
	sub.l	d5,d1		; (Y1-Y3)=Vy1
	sub.l	d4,d2		; (X2-X3)=Vx2
	sub.l	d5,d3		; (Y2-Y3)=Vy2

	muls.w	d3,d0		; Vx1*Vy2
	muls.w	d2,d1		; Vy1*Vx2

	sub.l	d1,d0		; (Vx1*Vx2)-(Vy1*Vy2)
	movem.l	(sp)+,d1-d6
	rts
